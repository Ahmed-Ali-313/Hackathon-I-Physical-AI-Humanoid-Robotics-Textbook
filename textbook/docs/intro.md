# Welcome to Physical AI & Humanoid Robotics

Welcome to the comprehensive course on **Physical AI & Humanoid Robotics**. This textbook will guide you through the complete robotics stack, from low-level control systems to high-level AI integration.

## What You'll Learn

This course covers the entire pipeline for building intelligent humanoid robots:

### 🤖 Module 1: ROS 2 - The Robotic Nervous System
Master the Robot Operating System 2, the industry-standard middleware for robotics. Learn about:
- Middleware architecture and DDS communication
- Nodes, topics, services, and actions
- Python-ROS bridging for AI integration
- URDF modeling for humanoid robots

### 🎮 Module 2: Digital Twin Simulation
Build photorealistic virtual environments for robot testing:
- Physics simulation with Gazebo and Unity
- Photorealistic rendering and interaction
- Sensor simulation (cameras, LiDAR, IMU)
- Virtual-to-real transfer techniques

### ⚡ Module 3: NVIDIA Isaac Platform
Harness GPU-accelerated robotics with NVIDIA's ecosystem:
- Isaac Sim for GPU-accelerated simulation
- Isaac ROS for hardware-accelerated perception
- Nav2 for autonomous navigation and planning
- Real-time performance optimization

### 🧠 Module 4: Vision-Language-Action Models
Integrate cutting-edge AI for intelligent robot control:
- LLM integration with GPT-4 and Claude
- Voice-to-action with Whisper speech recognition
- Cognitive planning and multi-step reasoning
- Building an autonomous assistant (capstone project)

## Hardware Requirements

To get the most out of this course, you'll need:

- **Workstation**: NVIDIA RTX GPU (12GB+ VRAM recommended), Ubuntu 22.04 LTS
- **Edge Kit** (optional): NVIDIA Jetson Orin Nano, Intel RealSense cameras
- **Robot Hardware** (optional): Unitree Go2 (quadruped) or Unitree G1 (humanoid)

Don't worry if you don't have all the hardware - most exercises can be completed in simulation!
